## Session Workflow

1. Upload `session-log.md` (and any other changed docs) to the Claude Project
2. Tell Claude: *"New session. Docs uploaded. Let's pick up where we left off."*
3. Play the session
4. At session end, ask Claude to generate the recap
5. Paste recap into `session-log.md`, update Current State block, commit
